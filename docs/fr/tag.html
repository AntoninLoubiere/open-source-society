Étiquettes

La liste des étiquettes :

  • Acteur: Les acteurs (fondations, entreprise privées) de l’open-source.
    • Fondation: Les acteurs qui sont des fondations.
    • Entreprise privée: Les acteurs qui sont des entreprises privées.
  • Projet: Les projets open-source.
    • Application: Les projets qui sont des applications.
    • Ordinateur: Les projets qui sont disponibles sur ordinateur.
      • Windows: Les projets qui sont disponibles sur Windows.
      • Linux: Les projets qui sont disponibles sur Linux.
      • Mac OS: Les projets qui sont disponibles sur MacOS.
    • Portable: Les projets qui sont disponibles sur portable.
      • Android: Les projets qui sont disponibles sur Android.
      • iOS: Les projets qui sont disponibles sur iOS.
    • Bureautique: Les projets de bureautique.
      • Navigateur: Les navigateurs.
  • Licence: Les licences des projets open-source.
