Étiquettes

Les étiquettes utilisé sur ce site :

  • Acteur: Un acteur (fondation, entreprise privée) de l’open-source.
    • Fondation: Un acteur qui est une fondation.
    • Entreprise privée: Un acteur qui est une entreprise privée.
  • Licence: Une licence de projets open-source.
  • Projet: Un projet open-source.
    • Application: Une application.
      • Ordinateur: Un projet qui est disponible sur ordinateur.
        • Windows: Un projet qui est disponible sur Windows.
        • Linux: Un projet qui est disponible sur Linux.
        • Mac OS: Un projet qui est disponible sur MacOS.
      • Portable: Un projet qui est disponible sur un portable.
        • Android: Un projet qui est disponible sur Android.
        • iOS: A project that is available on iOS.
    • Bureautique: Un projet de bureautique.
      • Navigateur: Un navigateur.
    • Communication: Un projet de télécommunication.
      • Messagerie Instantanée: Un projet de communication instantanée.
